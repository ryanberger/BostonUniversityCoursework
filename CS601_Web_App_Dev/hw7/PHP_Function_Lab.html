CS 601

Functions in PHP

Objectives

The objectives for this laboratory are solely intended to have you gain hands-on experience declaring and using functions in PHP.

Prerequisites

Students should have either Apache 2.2.x web server installed with the PHP extensions enabled, or have IIS 5.0 or greater installed with PHP's DLLs installed and configured.

Laboratory Details

Create solutions for each question:

  1. Create 4 functions:
    • renderHeader(): This function returns a div element with a BU logo on the left-hand side
    • renderMenu(): This function returns a div with a ul list of anchor elements. All links point to "#"
    • renderContent(): This function will contain all your content for the page(s).
    • renderFoot(): This shows the Copyright © 2009 [you name here]
  2. Create a function and a HTML page that shows a textfield and a submit button. Have the user indicate an age (any valid age...). The PHP function will calculate their age in dog years.
  3. Building off the previous example, have your function use a default value of 1 - just in case the user doesn't put in an integer value.
  4. Building off the previous example, include client-side validation that will check for non-numeric values before letting the form submission through to the server.
  5. Create a function called calculateTaxes(), that takes in a monetary figure. That function will call addStateTax, using that value. The function addStateTax will return a value with the tax included. Next, create and call another function called addFederalHealthCareExciseTax(). That does the same thing. Next, call another function that will will add salesTax. Have a textfield and submit button that interacts with the user. Show the results of the taxes in a div under the textfield.
